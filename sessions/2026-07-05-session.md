## What We Found

- **BLOCKER (U8):** MusicBrainz contains **bootleg/mashup recordings** that co-credit artists who never collaborated. The Beatles connected to Kendrick via a Kanye×Beatles mashup bootleg ("What's a Black Beatle?", status=Bootleg) — contradicting the plan's validated "Beatles is isolated" premise (that was a 100-recording sampling artifact). Escalated → user chose the **Official-release filter**.
- Official filter is necessary but **not fully sufficient**: novelty/**troll** tracks are marked Official and slip through — "Friday Part 3" by "Hanging Dong" (credits McCartney/Kanye/Kendrick/Cher) bridges McCartney→Kendrick, so The Beatles is still reachable via McCartney. No clean structural signal → user chose **accept + report honestly** (no hand-blocklisting).
- **DJ-mix false edges:** continuous-mix releases store blended segments as one recording co-crediting both artists ("Sweet Love / wacced out murals" credited "Anita Baker / Kendrick Lamar" on a DJ-mix album). This is a **clean structural signal** (release-group secondary type 8) → excluded.
- **Wrong-preview bug:** `get_preview` had a lenient "first result with a preview" fallback → served JP Saxe's "If the World Was Ending" for Earl Sweatshirt's "Really Doe". Also same-titled-wrong-artist risk (Ice Cube's "Really Doe").
- **HTML-render bug:** `display_path` split HTML across multiple `st.markdown` calls with unclosed tags; current Streamlit escapes that → raw HTML leaked as text.
- **Coverage (U6, final DJ-mix-filtered graph):** 14,881 artists / 38,377 edges. McCartney resolves (deg 1 via novelty track, or deg 2 via Kanye). Beatles reachable deg 2 (accepted caveat). ~0% no-connection across famous artists; the "Kanye West" miss is an **alias artifact** (MB canonical name is "Ye"). Frank Sinatra & David Bowie dropped after DJ-mix filter (were mix artifacts).
- Spotify DB (`data/collaboration_network.db`) confirmed **byte-for-byte untouched** (md5 `804a9fce...`) across all rebuilds.

## Decisions Made

- **Graph source = MusicBrainz dump** (CC0, no rate limit); Spotify crawl retained as fallback (KTD4). Previews = iTunes+Deezer (Spotify `preview_url` deprecated).
- **Edge = shared recording co-credited on an Official release only** (KTD10). Nodes keyed on MBID (KTD7). Versions dedup to a canonical song (KTD8). Depth 2 only this iteration (KTD5).
- **Exclude release-group secondary types Interview/Spokenword/Audiobook + DJ-mix** from edges. DJ-mix added after finding blend artifacts — a clean structural signal, so it's principled filtering, not curation.
- **Novelty/troll Official tracks (e.g. "Friday Part 3"): accept + report honestly, do NOT hand-blocklist.** Consequence: "Beatles returns no connection" is NOT met; recorded as an accepted MusicBrainz-data caveat.
- **Previews: correctness over coverage** — require title+artist match; degrade to a search link rather than ever show a wrong track.
- **Next focus = DATA QUALITY** (per user): continue the data-bug work — **depth-3 expansion** and **artist-alias search** — BEFORE design or public deployment. Deployment deferred; branch not pushed / no PR yet.
